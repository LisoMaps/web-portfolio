AGOL FL Field Population

This ArcPy script tool automatically populates and manages the ‘IDENTIFIER’ field in AGOL feature layers, ensuring the consistency and proper formatting of newly populated ‘IDENTIFIER’ field values.

Our client wanted to automate a vector tile package (.vtpk) creation workflow. After creation, the .vtpk's needed to be uploaded to Azure blob storage so they could be retrieved by an API and ingested into their new vegetation management app. Thus this suite of script tools was born!
